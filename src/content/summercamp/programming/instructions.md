Firstly, Welcome to DHS Robotics! We hope you'll have a good time here. 

## Course Outline

This course will go through the very very basics of programming. 
At the time of writing, this is the first time this course has been used. 
If anything seems missing or unclear, please seek help from a _qualified individual_.
Ideally, these problems will get ironed out as time goes on.

This course will teach you how to install necessary software, then how to control the robot with a physical controller. Finally, the summer camp course will end with the basics for autonomous control. 

## Note from the Author:

Historically, I have been an awful teacher. I do not know what pacing is best for this course.
If you feel lost, overwhelmed, or tired, please work through it. Learning programming is not an easy or quick process, take breaks and don't give up! Good luck future world champion. :smile:
