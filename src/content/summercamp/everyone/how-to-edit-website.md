## 1: Clone the Repository

```bash
git clone https://github.com/DHRA-2131/website
```

## 2: Open In VSC

Open the website in VSC. Do this the same way you would a pros project.

## 3: Add / Edit / Remove Files

You can add, remove or edit files in the `src/content/programming`, `src/content/learn`, or `src/content/summercamp` directories. 
Adding a new catagory not be covered in this article. Please visit [Astro Documentation](https://astro.build/).

## 4: Updating the Config

If you removed or added a file, go to `src/config.ts`. Add / Remove the file from the `LEARN_SIDEBAR`, `PROGRAMMING_SIDEBAR`, or `SUMMER_CAMP_SIDEBAR`. 

## 5: Verify Build

This will require npm to be installed on your laptop (As of time of writing, npm cannot be installed easily on a school laptop). 

```bash
npm run dev
```

This will host a local website on [Local Host, Port: 4321](http://localhost:4321/website/). 

## 6: Push to Github

This should be easy at this point! Visit [Using Source Control](../../../programming/getting-started/using-source-control/ "mention").
